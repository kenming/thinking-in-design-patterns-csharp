# 物件導向基礎範例專案

## 概述
本專案包含書籍 **Ch01** 的物件導向程式設計基礎範例，涵蓋OOP核心概念與SOLID設計原則。

本專案旨在透過實際的程式碼範例，展示物件導向程式設計的三個核心概念：關聯關係 (Association)、整體-局部關係 (Whole-Parts) 以及一般化-特殊化關係(Generalization-Specialization)。專案使用 C# 語言實作，並以汽車相關的類別作為範例。

## 專案結構

```
OOFundamentalSolution/
└── OOFundamental/
    ├── Program.cs                        # 主程式入口點，提供互動式選單
    ├── 1.Association/                    # 關聯關係範例
    │   ├── Car.cs                        # 車輛類別，展示屬性和行為的定義
    │   └── Driver.cs                     # 駕駛員類別，展示與車輛的關聯關係
    ├── 2.Whole-parts/                    # 整體-部分關係範例
    │   ├── Car.cs                        # 整體類別，包含引擎和輪胎組件
    │   ├── Driver.cs                     # 駕駛員類別，展示與車輛的關聯
    │   ├── Engine.cs                     # 引擎類別，作為車輛的組件
    │   └── Tire.cs                       # 輪胎類別，作為車輛的組件
    └── 3.Generalization-Specialization/  # 一般化-特殊化關係範例
        ├── Car.cs                        # 車輛抽象基類，定義共同屬性和行為
        ├── CarFactory.cs                 # 車輛工廠類別，負責建立不同類型車輛
        └── Sedan.cs                      # 轎車類別，Car的子類別
```

## 學習重點

- 關聯 (Association) - 物件間的連結關係
- 整體-局部 (Whole-Parts) - 組合與聚合關係
- 一般化-特殊化 (Generalization-Specialization) - 繼承與多型
- 抽象類別與方法
- 工廠模式實作
